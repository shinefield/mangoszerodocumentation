Database Guide
==============

*mangos-zero* stores all game data and session data such as accounts, or the
state of the game within a database of your choice, such as `MySQL`_ or
`PostgreSQL`_.

.. _MySQL: http://www.mysql.com/
.. _PostgreSQL: http://www.postgresql.org/

You will learn more about the database structure, and how the parts are connected
