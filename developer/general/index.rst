General Guide
=============

*mangos-zero* is developed using *C++*, and is fully compliant with `C++03`_.
Partially we employ `C++11`_ features on compiler architectures supporting it.

.. _C++03: http://en.wikipedia.org/wiki/C++03
.. _C++11: http://en.wikipedia.org/wiki/C++11

You will learn more about the project architecture, coding standards, style
issues, and see how changes are supposed to be done properly.
