===============
ItemPetFood.dbc
===============

The *item pet food* table contains definitions for pet food types.

Table structure
---------------

+------+--------+----------------------+-----------+----------------------------------+
| ID   | Name   | Type                 | Default   | Description                      |
+======+========+======================+===========+==================================+
| 1    | ID     | Integer (signed)     | -         | Unique ID                        |
+------+--------+----------------------+-----------+----------------------------------+
| 2    | name   | String (localized)   | -         | The name of the pet food type.   |
+------+--------+----------------------+-----------+----------------------------------+
