=============
QuestInfo.dbc
=============

The *quest info* table contains definitions for quest types.

Table structure
---------------

+------+------------+----------------------+-----------+-------------------------------+
| ID   | Name       | Type                 | Default   | Description                   |
+======+============+======================+===========+===============================+
| 1    | ID         | Integer (signed)     | -         | Unique ID                     |
+------+------------+----------------------+-----------+-------------------------------+
| 2    | name       | String (localized)   | -         | The name of the quest type.   |
+------+------------+----------------------+-----------+-------------------------------+
