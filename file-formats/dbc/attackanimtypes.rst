===================
AttackAnimTypes.dbc
===================

The *attack animation types* table contains definitions for attack
animation types.

Table structure
---------------

+------+--------+----------------------+-----------+-----------------------------------+
| ID   | Name   | Type                 | Default   | Description                       |
+======+========+======================+===========+===================================+
| 1    | ID     | Integer (signed)     | -         | Unique ID                         |
+------+--------+----------------------+-----------+-----------------------------------+
| 2    | name   | String (localized)   | -         | The name of the animation type.   |
+------+--------+----------------------+-----------+-----------------------------------+

Data
----

+------+--------------------+
| ID   | Name               |
+======+====================+
| 1    | 1H\_Main\_Swing    |
+------+--------------------+
| 2    | 1H\_Main\_Pierce   |
+------+--------------------+
| 3    | 2HL\_Pierce        |
+------+--------------------+
| 4    | 2HL\_Swing         |
+------+--------------------+
| 5    | 2HT\_Swing         |
+------+--------------------+
| 7    | OffH\_Swing        |
+------+--------------------+
| 8    | OffH\_Pierce       |
+------+--------------------+
