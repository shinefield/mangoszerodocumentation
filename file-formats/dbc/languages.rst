=============
Languages.dbc
=============

The *languages* table contains definitions for in-game languages.

Table structure
---------------

+------+--------+----------------------+-----------+----------------------+
| ID   | Name   | Type                 | Default   | Description          |
+======+========+======================+===========+======================+
| 1    | ID     | Integer              | -         | Unique ID            |
+------+--------+----------------------+-----------+----------------------+
| 2    | name   | String (localized)   | -         | The language name.   |
+------+--------+----------------------+-----------+----------------------+
