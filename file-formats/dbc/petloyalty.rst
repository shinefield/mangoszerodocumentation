==============
PetLoyalty.dbc
==============

The *pet loyalty* table contains definitions for pet loyalty levels.

Table structure
---------------

+------+--------+----------------------+-----------+----------------------------------+
| ID   | Name   | Type                 | Default   | Description                      |
+======+========+======================+===========+==================================+
| 1    | ID     | Integer (signed)     | -         | Unique ID                        |
+------+--------+----------------------+-----------+----------------------------------+
| 2    | name   | String (localized)   | -         | The name of the loyalty level.   |
+------+--------+----------------------+-----------+----------------------------------+
