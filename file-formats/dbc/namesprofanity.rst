==================
NamesProfanity.dbc
==================

The *names profanity* table contains definitions for characters names
which are disallowed.

Table structure
---------------

+------+--------+--------------------+-----------+-------------------------------------+
| ID   | Name   | Type               | Default   | Description                         |
+======+========+====================+===========+=====================================+
| 1    | ID     | Integer (signed)   | -         | Unique ID                           |
+------+--------+--------------------+-----------+-------------------------------------+
| 2    | name   | String             | -         | The name considered as profanity.   |
+------+--------+--------------------+-----------+-------------------------------------+
