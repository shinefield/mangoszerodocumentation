=====================
TerrainTypeSounds.dbc
=====================

The *terrain type sounds* table contains definitions for sounds
available for each terrain type.

Table structure
---------------

+------+--------+--------------------+-----------+---------------+
| ID   | Name   | Type               | Default   | Description   |
+======+========+====================+===========+===============+
| 1    | ID     | Integer (signed)   | -         | Unique ID     |
+------+--------+--------------------+-----------+---------------+
