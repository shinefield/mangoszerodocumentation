===============
DBC File Format
===============

The `World of Warcraft`_ game client contains a few local database files, which
are used to bootstrap the client and provide a set of common data which is used
in all game data transferred between client and server.

This section describe the file format itself, and covers the structure for each
**DBC** file included the the game client.
