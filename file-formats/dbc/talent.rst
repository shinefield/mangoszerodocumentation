==========
Talent.dbc
==========

The *talent* table contains definitions for all available class talents
and their requirements.

Relations
---------

-  ``tabID`` references the primary key of ``TalentTab.dbc``
-  ``spellRank[1-9]`` and ``requiredSpellId`` reference the primary key
   of ``Spell.dbc``
