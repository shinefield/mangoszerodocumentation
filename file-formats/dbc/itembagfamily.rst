=================
ItemBagFamily.dbc
=================

The *item bag family* table contains definitions for item bag families.

Table structure
---------------

+------+--------+-----------------------+-----------+------------------------------------+
| ID   | Name   | Type                  | Default   | Description                        |
+======+========+=======================+===========+====================================+
| 1    | ID     | Integer               | -         | Unique ID                          |
+------+--------+-----------------------+-----------+------------------------------------+
| 2    | name   | String (localized)    | -         | The name of the item bag family.   |
+------+--------+-----------------------+-----------+------------------------------------+
