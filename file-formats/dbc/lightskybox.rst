===============
LightSkybox.dbc
===============

The *light skybox* table contains definitions for models used as sky
box.

Table structure
---------------

+------+--------+--------------------+-----------+------------------------------+
| ID   | Name   | Type               | Default   | Description                  |
+======+========+====================+===========+==============================+
| 1    | ID     | Integer (signed)   | -         | Unique ID                    |
+------+--------+--------------------+-----------+------------------------------+
| 2    | name   | String             | -         | Path to the sky box model.   |
+------+--------+--------------------+-----------+------------------------------+
